# Discord Setup

Server name:
SelfHostGuard

## Channels

- `#announcements` - project updates, releases, and important notices.
- `#general` - general self-hosting and BackupProof discussion.
- `#backupproof` - focused discussion about BackupProof usage, rules, and roadmap.
- `#support` - help with running the CLI and understanding reports.
- `#feature-requests` - ideas for new checks, outputs, and integrations.
- `#bugs` - bug reports and reproduction discussion. Remind users not to paste secrets.
- `#contributors` - contributor coordination, PR discussion, and docs work.

## Manual Setup Notes

- Create the server manually.
- Add a short rules message that asks users not to paste secrets, tokens, private hostnames, or full production Compose files.
- Pin the GitHub repo link in `#backupproof` and `#support`.
- Keep early moderation simple.
- Do not announce the Discord widely until the GitHub repo has enough docs to handle first-time users.
